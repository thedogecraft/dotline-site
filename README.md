# Dotline Site

Landing page for [Dotline](https://github.com/Parcoil/dotline), an open-source crosshair overlay for Windows, Linux, and macOS.

Built with Next.js, Tailwind CSS, and shadcn.

## Development

```bash
pnpm install
pnpm dev
```

## Docker

```bash
docker compose up --build
```

Runs on `http://localhost:3426`.
